## Synopsis

Use the `DROP_REPLICATION_SLOT` statement to drop a replication slot.

## Syntax

{{%ebnf%}}
  drop_replication_slot
{{%/ebnf%}}

## Semantics

### *slot_name*

The name of the replication slot.

## Examples

- Drop a replication slot.

```plpgsql
yugabyte=# DROP_REPLICATION_SLOT test_replication_slot;
```
